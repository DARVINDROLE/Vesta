# Vesta: Your Personal Guide to the Cosmos

Vesta is an interactive web application designed to provide an immersive and educational experience about our solar system, asteroids, and the potential threats they pose. Explore the cosmos, learn about celestial bodies, and understand the importance of planetary defense.

## Features

*   **Augmented Reality (AR) and Virtual Reality (VR) Experience:**
    *   Explore a stunning 3D model of our solar system in an immersive VR environment.
    *   Bring celestial objects into your own world with our AR mobile integration. Point your device to the sky to identify stars and planets, or visualize an asteroid in your room.

*   **Asteroid Impact Simulator:**
    *   Launch your own virtual asteroids and witness their trajectory through the solar system.
    *   Experience the potential consequences of an impact event with our "Minecraft Extinction" simulation.

*   **Interactive Quiz Game:**
    *   Test your knowledge with our engaging quiz game. Learn fun facts about space and astronomy while competing for a high score.

*   **Trajectory Predictions:**
    *   Track near-earth objects and predict their future trajectories. Our application uses real-time data to provide accurate predictions.

*   **Mitigation Strategies Game:**
    *   Learn about different asteroid mitigation strategies through our interactive game. Make critical decisions to save the planet from a potential impact.
    *   Share your high scores and achievements on social media!

## How to Run

To run Vesta on your local machine, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Nakul03/Vesta.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd vesta
    ```
3.  **Install the dependencies:**
    ```bash
    npm install
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
This will start the application on your local server, usually at `http://localhost:5173`.
